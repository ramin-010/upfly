Upfly – npm package for Express image upload & conversion

Upfly is an Express middleware that uploads files and converts images with Multer and Sharp. Convert to webp, jpeg, png, or avif; keep buffers in memory or save to disk.

npm · GitHub

Install

npm i upfly multer
# or
yarn add upfly multer
# or
pnpm add upfly multer

Quick start

const express = require('express');
const { upflyUpload } = require('upfly');

const app = express();

app.post(
  '/upload',
  upflyUpload({
    fields: {
      images: { output: 'memory', format: 'webp', quality: 80 },
    },
    outputDir: './uploads',
  }),
  (req, res) => {
    res.json({ files: req.files, file: req.file });
  }
);

Path safety

Paths like /uploads or \uploads resolve under your project root. Use explicit absolute paths to write outside the project (e.g., C:\data\uploads or /var/data/uploads).
